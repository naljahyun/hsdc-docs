# 내원 전 필요한 안내를 한곳에서 확인하세요
<div class="doc-meta">철저한 위생, 세심한 배려, 생애주기를 고려한 치료를 지향합니다.</div>

<div class="page-home" markdown>

환자분이 서류 발급, 보험 청구, 비급여 비용 확인을 위해 다시 문의하지 않아도 되도록 정리한 안내 페이지입니다. 필요한 항목을 먼저 확인하시고, 실제 발급 및 비용 확정은 내원 시 데스크에서 안내받아 주세요.

<h2 class="home-section__title">먼저 확인하면 좋은 순서</h2>
</div>

<ol class="home-steps" markdown>

<li>발급받을 서류가 정해졌다면 제증명서 발급 비용에서 예상 비용을 확인합니다.</li>
<li>치료 비용 자체가 궁금한 경우 비급여 수가표에서 항목별 기준가를 확인합니다.</li>
